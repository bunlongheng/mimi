1
Clone: git clone https://github.com/bunlongheng/mimi ~/Sites/mimi
2
Install: brew install whisper-cpp ffmpeg
3
Model: cd ~/Sites/mimi && mkdir -p models && curl -L "https://huggingface.co/ggerganov/whisper.cpp/resolve/main/ggml-tiny.en.bin" -o models/ggml-tiny.en.bin

5

6
Auto-start: cp com.mimi.captions.plist ~/Library/LaunchAgents/ && launchctl load ~/Library/LaunchAgents/com.mimi.captions.plist
